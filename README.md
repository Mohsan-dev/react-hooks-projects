## 🪝 Hooks Used

- 🧠 **`useState`** — managing local UI state (inputs, toggles, filters)
- ⚡ **`useEffect`** — handling side effects like data fetching & syncing
- 📌 **`useRef`** — DOM access & persisting values without re-renders

## 🛠️ Getting Started

```bash
# Clone the repo
git clone https://github.com/Mohsan-dev/react-hooks-projects.git

# Navigate into any project folder
cd react-hooks-projects/weather-app

# Install dependencies
npm install

# Run the app
npm start
```
